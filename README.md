# 🎸 Chordao

Guitar chord visualizer based on E/Em/A/Am shape derivation.

Pick a key, see all 6 diatonic chords (I, IIm, IIIm, IV, V, VIm) across the fretboard — with the optimal movement path highlighted.

## Features

- Derive all chords from 4 base shapes (E / Em / A / Am) via barre transposition
- Full fretboard overview with degree-colored dots
- Simplified shape grid (2 rows × 12 frets)
- Auto-highlight the most efficient chord combination

## Dev

```bash
bun install
bun run dev
```

## Stack

React + TypeScript + Vite + Bun
